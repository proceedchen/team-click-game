# 阿里云部署指南

## 准备工作

确保你已购买：
- 阿里云 OSS（存储静态网页）
- 阿里云 Redis（存储排行榜数据，已有）
- 域名（可选，用于钉钉扫码）

---

## 第一步：创建 OSS Bucket

1. 登录阿里云控制台 https://oss.console.aliyun.com/
2. 点击「创建 Bucket」
3. 配置：
   - **Bucket 名称**：`data-dancer`（或其他名称）
   - **区域**：选择「华北 2」或离你近的区域
   - **存储类型**：标准存储
   - **读写权限**：公共读（这样可以直接访问）

4. 创建成功后，进入 Bucket

5. **上传 index.html**：
   - 点击「文件管理」→ 「上传文件」
   - 选择 `aliyun-deploy/index.html`
   - 注意：上传到根目录

6. **设置静态页面**：
   - 点击「数据管理」→ 「静态页面」
   - 设置「默认首页」为 `index.html`
   - 设置「默认 404 页」为 `index.html`

---

## 第二步：部署函数计算 API

1. 登录阿里云函数计算 FC：https://fc.console.aliyun.com/

2. 创建服务：
   - 点击「服务」→ 「创建服务」
   - 服务名称：`data-dancer-api`
   - 其他默认

3. 创建函数：
   - 进入刚创建的服务
   - 点击「创建函数」→ 「使用内置 runtime 创建」
   - 配置：
     - **函数名称**：`api`
     - **运行环境**：Python 3.10
     - **函数入口**：`index.handler`
     - **代码上传方式**：在线编辑

4. 复制代码：
   - 打开 `aliyun-deploy/index.py` 的内容
   - 粘贴到函数代码编辑器中
   - 点击「部署」

5. 配置触发器（可选）：
   - HTTP 触发器需要公网访问
   - 建议使用「自定义域名」绑定

---

## 第三步：配置自定义域名（推荐）

1. 进入函数计算「自定义域名」
2. 点击「创建域名」
3. 配置：
   - **域名**：填写你的域名（如 game.yourdomain.com）
   - **协议**：HTTPS（需要上传 SSL 证书）
   - **函数**：选择刚才创建的函数

4. 在你的域名服务商处添加 CNAME 记录：
   - 主机记录：game
   - 记录值：函数计算提供的域名

---

## 第四步：修改 index.html 的 API 地址

部署函数计算后，需要修改 index.html 中的 API 地址：

```javascript
// 找到这两行，替换为你的函数计算地址
const res = await fetch('/api/leaderboard');
// 改为
const res = await fetch('https://your-function-url/api/leaderboard');
```

或者如果你配置了自定义域名，API 会自动指向同源。

---

## 第五步：测试

- 访问 OSS 绑定的域名或自定义域名
- 应该能看到游戏页面
- 输入昵称开始游戏
- 排行榜数据会存入你的阿里云 Redis

## 注意事项

1. **Vercel 和阿里云数据不互通** - Vercel 用的是 Upstash Redis，阿里云用的是你的阿里云 Redis
2. **钉钉扫码** - 需要 HTTPS 域名才能在钉钉中打开
3. **备案** - 如果用国内域名，需要完成 ICP 备案
