# 数据点击器

一个适合钉钉扫码打开的 H5 点击小游戏：顶部显示累计点击数，中部偏下是大按钮，点击后会在屏幕中间弹出数据相关术语。

## 使用方式

1. 直接用手机浏览器或钉钉扫码打开 `index.html` 对应的线上地址。
2. 连续点击按钮，计数器会递增，术语会随机浮现。
3. 适合在活动页、群分享页、内部培训页中作为轻量互动玩法。

## 部署说明

这是一个纯静态页面，不依赖后端。把整个项目部署到任意静态站点即可。

### 方案一：本地快速预览

如果你只是想先看效果，可以直接用本地静态服务器打开项目目录，例如：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080/`。

### 方案二：发布到线上

可部署到以下任意静态托管平台：

- Vercel
- Netlify
- GitHub Pages
- 阿里云 OSS 静态网站
- 腾讯云 COS 静态网站

部署时只需要确保入口文件是 `index.html`。

### 钉钉扫码注意事项

- 线上链接要是可公开访问的 HTTPS 地址。
- 页面尽量保持单文件或少量静态资源，避免首次打开太慢。
- 如果要在钉钉里分享，建议用短链接或二维码指向最终页面地址。
